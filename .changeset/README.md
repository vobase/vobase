# Changesets

Hello and welcome! This folder has been automatically generated by `@changesets/cli`, a build tool that works
with multi-package repos, or single-package repos to help you version and publish your code. You can
find the full documentation for it [in the repository](https://github.com/changesets/changesets).

## How to add a changeset

1. Run `bunx changeset` in the repo root
2. Select the packages you want to include in the changeset
3. Choose the bump type (major, minor, patch)
4. Write a summary of the changes

The changeset will be committed as a markdown file in this directory. When the "Version Packages" PR is merged,
the changesets will be consumed and versions/changelogs will be updated automatically.
