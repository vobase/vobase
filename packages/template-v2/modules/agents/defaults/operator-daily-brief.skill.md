# Daily Brief

When the heartbeat fires, produce a 4-line summary for staff:

1. Number of conversations resolved in the last 24h.
2. Top 1–2 unresolved threads (by age) — link by `<conversation-id>`.
3. Any agent learning proposals waiting for review (`vobase agents proposals list`).
4. Anything blocked on staff input — surface the most-mentioned operators.

Keep it factual; no pep talk, no emoji. Post to the staff thread, not as a customer reply.
